# 📸 Instagram Feed Plugin

Display your Instagram posts on your Framer site with a beautiful, customizable grid.

## What It Does

**Instagram Feed** lets you:
- Connect your Instagram account
- Select which posts to display
- Customize the grid layout and appearance
- Embed a live feed on your Framer site

## How to Use

### 1. Install
1. Download `plugin.zip` from this repo
2. Go to [Framer Marketplace Dashboard](https://www.framer.com/marketplace/dashboard/plugins/)
3. Click "Add Plugin" → Upload `plugin.zip`

### 2. Connect Your Account
1. Open the plugin in Framer
2. Click **Connect Instagram**
3. Authorize the app (read-only access)
4. Your posts will load automatically

### 3. Select Posts
1. Browse your posts in the grid
2. Click to select/deselect posts
3. Use "Select All" for quick selection

### 4. Customize
- **Columns** — 2, 3, or 4 columns
- **Gap** — Space between images
- **Border Radius** — Rounded corners
- **Layout** — Grid or Masonry
- **Hover Effect** — None, Zoom, or Overlay
- **Show Captions** — Toggle caption display
- **Show Likes** — Toggle like counts

### 5. Preview & Export
1. See a live preview of your feed
2. Copy the embed code
3. Add to your Framer site

## Features

- ✅ **OAuth connection** - Secure Instagram login
- ✅ **Post selection** - Choose which posts to show
- ✅ **Grid layouts** - 2, 3, or 4 columns
- ✅ **Masonry option** - Pinterest-style layout
- ✅ **Custom styling** - Border radius, gaps
- ✅ **Hover effects** - Zoom, overlay
- ✅ **Caption support** - Show/hide captions
- ✅ **Like counts** - Display engagement
- ✅ **Live preview** - See changes in real-time

## Technical Notes

- Uses Instagram Basic Display API
- OAuth 2.0 for secure authentication
- Read-only access (no posting)
- Tokens refresh automatically

## Requirements

- Framer account
- Instagram account (public posts)
- Framer project
