# 📸 Instagram Feed

Display your Instagram posts on your Framer site with a beautiful, customizable grid.

## What It Does

- **OAuth Connection** — Secure Instagram login
- **Post Selection** — Choose which posts to display
- **Grid Layouts** — 2, 3, or 4 columns
- **Masonry Layout** — Pinterest-style option
- **Custom Styling** — Border radius, gaps, hover effects
- **Live Preview** — See changes in real-time
- **Dark/Light theme** — Automatic based on Framer settings

## How to Install

1. Download `plugin.zip` from this repo
2. Go to [Framer Marketplace Dashboard](https://www.framer.com/marketplace/dashboard/plugins/)
3. Click "Add Plugin" → Upload `plugin.zip`

## How to Use

### 1. Connect Account
Click **Connect Instagram** and authorize the app (read-only access).

### 2. Select Posts
Browse your posts and click to select which ones to display.

### 3. Customize
- **Columns** — 2, 3, or 4
- **Gap** — Space between images
- **Border Radius** — Rounded corners
- **Hover Effect** — Zoom or overlay

### 4. Preview & Add
Preview your feed and copy the embed code.
